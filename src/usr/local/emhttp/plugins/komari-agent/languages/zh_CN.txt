# Help text
Auto-start on boot and restart on crash.=开机自启,崩溃后自动重启。
Address of your Komari panel.=你的 Komari 面板地址。
Token: register one server with its token. Auto-discovery: register with a cluster key.=令牌:用服务器令牌注册单台。自动发现:用集群密钥注册。
Per-server token from the panel.=面板中对应服务器的令牌。
Cluster auto-discovery key from the panel.=面板中的集群自动发现密钥。
Block the panel from opening a web shell or running commands on this host.=阻止面板在本机开启 Web 终端或执行命令。
Let the agent update itself to new releases.=允许 Agent 自行更新到新版本。
Seconds between metric reports.=两次指标上报之间的秒数。
Skip TLS verification (for self-signed panel certificates).=跳过 TLS 校验(用于自签名面板证书)。
Extra command-line flags passed to the agent.=传给 Agent 的额外命令行参数。
Pin a version, or use latest.=固定某个版本,或使用 latest。
Optional GitHub mirror for binary downloads.=可选的 GitHub 镜像,用于下载二进制。
