# Help text
Auto-start on boot and restart on crash.=開機自動啟動，崩潰後自動重啟。
Address of your Komari panel.=你的 Komari 面板位址。
Token: register one server with its token. Auto-discovery: register with a cluster key.=權杖:以伺服器權杖註冊單台。自動探索:以叢集金鑰註冊。
Per-server token from the panel.=面板中對應伺服器的權杖。
Cluster auto-discovery key from the panel.=面板中的叢集自動探索金鑰。
Block the panel from opening a web shell or running commands on this host.=阻止面板在本機開啟 Web 終端或執行命令。
Let the agent update itself to new releases.=允許 Agent 自行更新到新版本。
Seconds between metric reports.=兩次指標回報之間的秒數。
Skip TLS verification (for self-signed panel certificates).=略過 TLS 驗證(用於自簽面板憑證)。
Extra command-line flags passed to the agent.=傳給 Agent 的額外命令列參數。
Pin a version, or use latest.=固定某個版本，或使用 latest。
Optional GitHub mirror for binary downloads.=可選的 GitHub 鏡像，用於下載二進位檔。
