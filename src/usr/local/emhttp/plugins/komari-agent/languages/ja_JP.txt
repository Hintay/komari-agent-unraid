# Help text
Auto-start on boot and restart on crash.=起動時に自動開始し、クラッシュ時に再起動します。
Address of your Komari panel.=Komari パネルのアドレス。
Token: register one server with its token. Auto-discovery: register with a cluster key.=トークン:サーバートークンで1台を登録。自動検出:クラスターキーで登録。
Per-server token from the panel.=パネルのサーバー別トークン。
Cluster auto-discovery key from the panel.=パネルのクラスター自動検出キー。
Block the panel from opening a web shell or running commands on this host.=パネルがこのホストで Web シェルを開いたりコマンドを実行するのを禁止します。
Let the agent update itself to new releases.=エージェントが新しいリリースへ自動更新するのを許可します。
Enable detailed GPU monitoring (usage, memory, multi-GPU).=詳細な GPU モニタリング（使用率、メモリ、マルチGPU）を有効にします。
Filter which network interfaces are monitored.=モニタリングするネットワークインターフェースをフィルタリングします。
Seconds between metric reports.=メトリクス送信の間隔(秒)。
Skip TLS verification (for self-signed panel certificates).=TLS 検証をスキップ(自己署名パネル証明書向け)。
Extra command-line flags passed to the agent.=エージェントに渡す追加のコマンドライン引数。
Pin a version, or use latest.=バージョンを固定、または latest を使用。
Optional GitHub mirror for binary downloads.=バイナリのダウンロードに使う任意の GitHub ミラー。
